🔒 Confidentialité

Les données de vos clients
restent chez vous

Contrairement aux outils SaaS classiques, CGP Skool ne possède aucun serveur de données. Tout se passe dans votre navigateur — c'est vous, et vous seul, qui conservez vos études.

✓Rien n'est envoyé sur Internet. Les données saisies ne quittent jamais votre navigateur.
✓Rien n'est conservé à votre insu. Fermez l'onglet : les données client sont effacées.
✓Vous archivez vous-même. La sauvegarde est un fichier JSON que vous téléchargez et rangez dans votre propre dossier client.
