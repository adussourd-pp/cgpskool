📋
Bienvenue sur l'Étude de Dossier
Voici comment utiliser l'outil en 5 étapes
1
Importer les données
Copie-colle le contenu du Word
▼
2 zones d'import dans la page :
• Recueil d'informations (RI) : ouvre le Word, Ctrl+A puis Ctrl+C, colle dans la zone et clique Valider. Le parser extrait identité, revenus, patrimoine, objectifs, fiscalité.
• Fiche légale (DER) : même principe, colle ton Document d'Entrée en Relation. Les habilitations ORIAS sont détectées automatiquement.
2
Vérifier & compléter
Corrige les erreurs du parser
▼
Après chaque import, un volet de vérification s'ouvre avec tous les champs pré-remplis. C'est la source de vérité : ce que tu valides ici alimente toute l'étude.

• Vérifie l'identité, la CSP, les revenus de chaque déclarant
• Ajoute ou supprime des actifs / passifs
• Coche les objectifs du client
• Renseigne l'épargne mensuelle et l'épargne de précaution
• Puis clique "Appliquer les modifications"
3
Parcourir l'analyse
Chaque section se remplit automatiquement
▼
Navigue avec les boutons Suivant / Précédent en bas de chaque page ou via la sidebar. L'ordre suit la pagination PDF :

Page de garde → Sommaire → Situation → Patrimoine → Impôts → Financement → Retraite → Préconisations → Synthèse → Mentions légales

Les sections Transmission, Entreprise et Avis clients sont optionnelles (activées par une case à cocher).
4
Sélectionner les solutions
Coche, paramètre, duplique
▼
Dans la section Préconisations, le panneau conseiller propose toutes les solutions (Copernic). Pour chaque solution :

• Cliquer = ajouter. Re-cliquer = retirer (si aucun montant saisi)
• "+ Ajouter un contrat" pour créer un 2e contrat pour un autre titulaire
• Renseigner les montants (VI, VP, horizon)
• Le libellé PDF est modifiable (ce que le client voit)
• Les recommandations cochées apparaissent en "Points de vigilance" dans le PDF
• La note client (switch) s'affiche sous la solution dans le PDF si activée
5
Exporter, sauvegarder & compte rendu
PDF, JSON, mails
▼
• Exporter PDF : génère l'étude complète. Après l'export, un popup propose de sauvegarder.
• Sauvegarder l'étude : télécharge un fichier JSON. Archive-le bien (Drive, Notion, dossier client) — c'est lui qui permet de reprendre l'étude plus tard.
• Charger une étude : sélectionne un JSON précédemment sauvegardé. Tout est restauré : données, solutions, notes.
• Compte rendu : 3 templates de mails (R1 Audit, R2 Présentation, R3 Confirmation) pré-remplis avec les données du client. Copie en un clic.
Astuces :
• Le fichier JSON sauvegardé contient TOUTE l'étude. Conserve-le précieusement pour pouvoir la reprendre 3, 6 ou 12 mois plus tard.
• Les mentions légales et l'avis d'imposition ont chacun leur propre zone de copier-coller + vérification dans Import / Saisie.
• Tu peux revoir ce guide à tout moment via "📖 Revoir le guide" en bas de la sidebar.
